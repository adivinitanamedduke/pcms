# Introduction 
This is a PCMS for managing product inventory through a web interface.
# Getting Started
Prerequisites
.NET 9.0 SDK or higher.
Node.js (v20.x or higher) & npm.
Angular CLI (npm install -g @angular/cli). 
# Backend Setup (.NET API) 
Navigate to the API folder:
bash
cd API

Trust the HTTPS Development Certificate:
(Required for the Angular app to communicate with the API securely)
bash
dotnet dev-certs https --trust

Restore and Build:
bash
dotnet restore
dotnet build

Run the API:
bash
dotnet run

The API will be available at: https://localhost:7260 (or your configured port).

#Frontend Setup (Angular SPA)
Navigate to the ClientApp folder:
bash
cd ClientApp
Use code with caution.

Install Dependencies:
bash
npm install

Configuration Check:
Ensure src/app/services/api.service.ts points to your active API URL.
Run the App:
bash
npm start

The UI will be available at: https://localhost:4200
Note: If you see a privacy warning, type thisisunsafe in the browser or "Proceed to localhost".

-API: PowerShell Certificate Script
This script creates a robust, password-protected certificate and imports it into your Trusted Root store to ensure the API is "Green" in the browser.
powershell
# Run as Administrator
$dnsName = "localhost"
$cert = New-SelfSignedCertificate -DnsName $dnsName -CertStoreLocation "cert:\LocalMachine\My" -NotAfter (Get-Date).AddYears(1)
$pwd = ConvertTo-SecureString -String "password123" -Force -AsPlainText

# Export to a PFX file for the API to use if needed
$path = "$PSScriptRoot\localhost-dev.pfx"
Export-PfxCertificate -Cert $cert -FilePath $path -Password $pwd

# Trust the certificate
Import-PfxCertificate -FilePath $path -CertStoreLocation "Cert:\LocalMachine\Root" -Password $pwd
Write-Host "SSL Certificate created and trusted for $dnsName" -ForegroundColor Green
Use code with caution.

-Client App: mkcert Setup
mkcert is a zero-config tool that creates locally-trusted development certificates. It is more reliable for Angular than standard self-signed certs. [3]
Install mkcert: choco install mkcert (Windows) or brew install mkcert (Mac).
Setup local CA: mkcert -install
Generate Certs for Angular:
Inside your ClientApp folder, run:
bash
mkcert localhost
Use code with caution.

This creates localhost.pem and localhost-key.pem.
Configure package.json:
Update your start script to use these specific files:
json
"start": "ng serve --ssl --ssl-cert localhost.pem --ssl-key localhost-key.pem"
